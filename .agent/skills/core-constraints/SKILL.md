# UI and Workflow Rules
A brief summary of what this skill covers (e.g., UI casing and browser behavior).

## Description
This skill provides specific constraints for UI text formatting and governs when the agent is allowed to use browser tools. Use this skill whenever generating UI components, writing copy, or performing environment checks.

## Instructions
* **Casing:** Always use "Sentence case" for all titles, headers, and labels in the UI. 
* **Browser Usage:** Never open the browser to verify things or search unless explicitly requested by the user.
